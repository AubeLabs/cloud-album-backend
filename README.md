## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

# cloud-album-backend

## Docker 빌드 및 실행

1. ### Docker 이미지를 빌드
```bash
docker build -t cloud-album-backend .
```
2. ### Docker Compose로 컨테이너 실행
```bash
docker-compose up -d
```
